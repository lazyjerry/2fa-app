# Secure 2FA 使用者操作指南

本文件說明日常操作流程：建立 vault、新增帳號、掃描 QR、複製驗證碼、鎖定。

## 安裝與啟動

1. 下載 macOS 應用程式封包後，開啟 Secure 2FA。
2. 首次啟動會進入建立 vault 畫面。
3. 之後每次啟動會進入解鎖畫面。

## 建立 Vault

1. 在主密碼欄位輸入至少 8 碼密碼。
2. 在確認密碼欄位輸入相同內容。
3. 送出後會建立本機加密 vault 並直接解鎖。

注意事項：
- 主密碼遺失無法復原，請自行妥善保存。
- 主密碼不會儲存在可回讀格式。

## 解鎖 Vault

1. 啟動 App 後輸入主密碼。
2. 點擊解鎖。
3. 若密碼錯誤，畫面會顯示錯誤訊息。

## 新增帳號

新增頁提供三種模式。

### 1) QR 圖片

1. 切到「新增」頁。
2. 選擇「QR 圖片」模式。
3. 上傳含有 otpauth URI 的 QR 圖片。
4. 確認 issuer、帳號名稱、分類與備註後儲存。

### 2) URI 貼上

1. 切到「新增」頁。
2. 選擇「URI」模式。
3. 貼上 otpauth:// 開頭的完整 URI。
4. 補上分類與備註後儲存。

### 3) 手動輸入 Secret

1. 切到「新增」頁。
2. 選擇「手動」模式。
3. 輸入 issuer、帳號名稱、Base32 secret。
4. 視需求調整演算法、位數與週期。
5. 儲存後回到驗證碼清單。

## 複製驗證碼

1. 在驗證碼清單找到目標帳號。
2. 點擊複製按鈕。
3. 程式會把目前一次性密碼寫入剪貼簿。
4. 到期前可在設定頁調整自動清空秒數。

剪貼簿保護行為：
- 到達倒數時，只有剪貼簿仍是本程式最後一次複製的驗證碼才會清空。
- 若你中途複製了其他內容，程式不會覆蓋你的新內容。

## 鎖定 Vault

1. 點擊右上角鎖定按鈕。
2. 程式會清除解鎖 session 並回到登入畫面。
3. 鎖定後無法讀取帳號清單與驗證碼。

另外，閒置超過設定秒數也會自動鎖定。

## 設定頁重點

- 遮蔽驗證碼：預設開啟，降低肩窺風險。
- 自動鎖定秒數：限制解鎖後暴露時間。
- 剪貼簿清空秒數：控制驗證碼在剪貼簿停留時間。
- 平台截圖防護：macOS 與 Windows 支援，切換後需重啟 App。

## 匯出與匯入

### 匯出

1. 進入設定頁的安全操作區。
2. 選擇匯出，輸入主密碼確認。
3. 選擇輸出檔案位置。

### 匯入

1. 進入設定頁的安全操作區。
2. 選擇匯入備份或批次匯入帳號。
3. 選檔後程式會自動檢查格式與重複項目。

## 常見問題

1. 解鎖一直失敗
   - 請先確認大小寫與輸入法。
   - 若 vault 檔案受損，程式會回報密碼錯誤或檔案損毀類型訊息。
2. QR 圖片無法匯入
   - 確認 QR 內容是 otpauth URI。
   - 重新截圖或改用 URI 貼上。
3. 截圖防護沒變化
   - 設定切換後需完整重啟 App 才會套用。
